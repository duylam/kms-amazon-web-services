== EC2

- Remote to machine (1), ping machine (2)
- Retrieve metadata:
	+ $ curl -i http://169.254.169.254/latest/user-data
	+ $ curl -i http://169.254.169.254/latest/meta-data
- Adjust instance type

== CLOUD WATCH

== lOAD BALANCING
- Access web via load balancing and observe the output from console in (3) instance: $ curl –i http://<balancing dns>/balacing


= AUTO SCALING
